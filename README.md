# Mobile Game Factory

Mobile Game Factory, haftada 1 basit mobil oyun üreten, her oyun için içerik paketi hazırlayan ve tüm agent süreçlerini dashboard üzerinden görünür kılan yarı otonom bir üretim sistemidir.

## Project Goal
- Haftada 1 mobil oyun üretmek
- Her oyun için YouTube ve kısa içerik paketi üretmek
- Agent bazlı görev dağılımı kurmak
- Dashboard üzerinden durum, blokaj ve review süreçlerini yönetmek

## Initial MVP Scope
- 1 oyun fikri seçimi
- 1 mini game design flow
- 1 prototype build task
- 1 content package task
- 1 dashboard görünümü
- 1 review inbox mantığı

## Project Structure
- `docs/` → planlar, blueprint, workflow, fazlar
- `dashboard/` → yönetim paneli
- `backend/` → Python tabanlı orchestration ve API katmanı
- `agents/` → agent tanımları ve görev mantığı
- `games/` → Unity oyun projeleri
- `prompts/` → agent prompt şablonları
- `content/` → video, thumbnail, store content taslakları
- `infra/` → docker ve teknik altyapı dosyaları

## Current Status
Phase 0 has started.

System blueprint and foundational project structure are being prepared.
